Recipes in Web Programming

Recipe 1:

Dynamically Changing Web Content

Ingredients:

  • The DOM createElement method
  • The DOM append method
  • Template literals
  • The innerText method
  • the innerHTML method

Instructions:

  1. Create a web document and fill with basic tags
  2. To the body of the document, add a div element of class demo
  3. Add script tags to the body of the document, just before the closing body tag
  4. Now use the DOM method, querySelector to get a reference to the div of class demo
  5. Use textContent method to add plain text
  6. Add some HTML code. Use innerHTML method
  7. Use the DOM createElement method, with append

Recipe 2:

Displaying JSON Data

Ingredients:

  • The DOM createElement method
  • The DOM append method
  • Array of objects
  • Choice of images

Instructions:

JSON stands for JavaScript Object Notation, and it comes close to an array of objects, like this array of animals below. Note, it is important to make sure that objects have common attributes, if these attributes are being used to display information.

Make sure that the images are referenced correctly, then you can use the techniques in Recipe #1 to display the animals on a web page. You can use a simple for or while, or array method forEach.
